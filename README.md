#### 从蛮牛下载的 参考博主的介绍

#UnityVLC

Unity调用VLC的库解码视频显示到3D场景中，支持常见本地视频和rtmp、rtsp等视频流。

本工程仅为Demo，做产品请各位商店寻找UMP插件

系统是64位的，因为我用的VLC的类库是64位的，用64位的Unity导出exe时选择x86_64，导出后需要将工程中Plugins下的plugins文件夹拷贝到发布后的Plugins文件夹下

示例是播放湖南卫视的rtmp视频流
